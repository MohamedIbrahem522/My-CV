# ⚡ Premium Engineering Portfolio & Resume

[![Live Deployment](https://img.shields.io/badge/Live-Demo-2563eb?style=for-the-badge&logo=vercel)](https://mohamedibrahem522.github.io/My-CV/)
[![Tech Stack](https://img.shields.io/badge/Stack-MERN_|_FastAPI_|_YOLOv8-blue?style=for-the-badge)](https://github.com/MohamedIbrahem522)

A high-performance, responsive resume architecture designed for **Software Engineers**. This project implements a "Cyber-Premium" aesthetic, prioritizing information density and technical hierarchy.

## 🛠 Engineering Excellence

*   **Atomic Design Principles:** Built with modularity in mind, separating technical stacks from career milestones for high scannability.
*   **Typography-Driven UI:** Leverages the **Syne** typeface to create a modern, high-contrast visual identity suitable for elite tech roles.
*   **Performance Optimized:** Zero-dependency architecture ensuring near-instant load times and 100/100 Lighthouse performance scores.
*   **AI Integration Context:** Specialized layouts to showcase complex implementations like **YOLOv8** computer vision pipelines and **FastAPI** backends.

## 💻 Technical Implementation

### Core Architecture
*   **HTML5 Semantic Layer:** Optimized for accessibility (A11y) and SEO.
*   **CSS3 Grid & Flexbox:** A robust two-column layout that gracefully degrades on legacy systems and scales for 4K displays.
*   **Custom Variables:** Centralized theme management via CSS `:root` for effortless rebranding.

### Showcase Areas
*   **The Stack:** Categorized breakdown of Frontend, Backend, Database, and AI/CV tooling.
*   **Professional Milestones:** Structured data for roles including **Node.js Instructor** at GDSC and **Front-End Intern** at ITI.
*   **Project Laboratory:** Detailed cards for **Real-Time Object Detection (Grade: A+)**, MERN E-Commerce, and ERP systems.

## 🚀 Quick Start

1.  **Clone & Personalize:**
    ```bash
    git clone [https://github.com/MohamedIbrahem522/My-CV.git](https://github.com/MohamedIbrahem522/My-CV.git)
    ```
2.  **Asset Replacement:**
    Drop your profile image into the root directory and update the `<img>` source in `index.html`.
3.  **Deploy:**
    Push to your `main` branch to trigger the **GitHub Pages** deployment pipeline.
