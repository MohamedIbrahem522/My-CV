سيرة ذاتية تفاعلية مصممة بدقة عالية لتعكس المهارات التقنية في تطوير الويب والذكاء الاصطناعي. تم بناء هذا المشروع ليكون واجهة احترافية تجمع بين البساطة (Minimalism) والقوة التقنية.

---

## 💎 ممتيزات المشروع (Elite Features)

- **Premium Typography**: استخدام عائلة خطوط **Syne** و **Syne Mono** لإعطاء طابع تقني حديث.
- **Grid-Based Architecture**: توزيع احترافي للمحتوى باستخدام **CSS Grid** لضمان أفضل تسلسل هرمي للمعلومات.
- **Interactive Experience**: بطاقات مشاريع تفاعلية مع تأثيرات (Hover Effects) تعزز من تجربة المستخدم.
- **Dark-Mode Compatible**: ألوان مريحة للعين تتماشى مع معايير التصميم الحديثة (Modern Ink & Paper).
- **Fully Responsive**: متوافق تماماً مع جميع الشاشات من الهواتف حتى شاشات الـ 4K.

---

## 🛠️ البناء التقني (Architecture)

### 💻 اللغات والأدوات
- **HTML5 Semantic Elements**: لضمان أفضل أداء لمحركات البحث (SEO).
- **Custom CSS Properties**: استخدام المتغيرات (Variables) لسهولة تعديل الألوان والهوية البصرية.
- **Mobile-First Workflow**: التصميم بدأ من أصغر شاشة لضمان الكفاءة.

### 🏗️ أقسام السيرة الذاتية
- **Stack Section**: عرض تقني منظم للـ Frontend والـ Backend وقواعد البيانات.
- **Experience**: تسلسل زمني احترافي لخبرات العمل كمدرب (Instructor) ومطور.
- **Projects Showcase**: عرض للمشاريع الكبرى مثل **YOLOv8 Object Detection** ومنصات الـ **E-Commerce**.
